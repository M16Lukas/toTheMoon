# toTheMoon(開発中)
 
## [ Introduction ]
Yahoo Finance APIを活用したアメリカ株検索＆コミュニティサイト開発中です。

## [ Verson ]

- Application Framework : Spring 4.3.6.RELEASE
- Persistence Framework : MyBatis 3.4.6
- Langauge : JAVA 8 (v 1.8)
- DateBase : Oracle 11g
